# homebrew-tap

[discord-assistant](https://github.com/Hyeonjun0527/discord-assistant) Provider Agent 용 Homebrew tap.

```bash
brew tap yeon-intergation-platform/tap
brew install discord-ai-provider-agent
```
`brew` 가 sha256 을 자동 검증하므로 수동 해시검증이 필요 없습니다. 일반 사용자 권한으로 실행됩니다.
릴리스마다 `discord-assistant` 의 CI(`agent-build.yml`)가 이 Formula 를 자동 갱신합니다.
